# Test Plan for NIDS Project

## Objective

Define how the NIDS project will be tested before final submission or deployment.

## Testing Scope

- Frontend testing
- Backend API testing
- Database testing
- ML model validation
- Security testing
- Performance testing

## 1. Frontend Testing

- Form validation
- Route navigation
- Error state rendering
- Dashboard rendering
- Alerts and reports filtering

## 2. Backend Testing

- Login success and failure
- Auth token validation
- API response format
- Prediction endpoint behavior
- Alert update logic
- Error handling paths

## 3. Database Testing

- Connection success
- Insert/update/delete operations
- Constraint validation
- Timestamp and index behavior

## 4. ML Testing

- Dataset preprocessing correctness
- Model training repeatability
- Evaluation metric reporting
- Inference response accuracy on test data

## 5. Security Testing

- Invalid login attempts
- Unauthorized route access
- SQL injection checks
- Malformed payload checks
- Rate limit validation

## 6. Performance Testing

- Prediction response time
- Dashboard load time
- Concurrent API request behavior
- Database query performance

## Deliverables

- Test cases
- Test result sheet
- Bug list
- Final validation summary
